# Croatia Guidebook

Контентный сайт-справочник по Хорватии на трёх языках (**en / ru / uk**): достопримечательности, города, еда, маршруты, транспорт, аренда авто, страховки, развлечения, релокация, новости. Полная статика на Astro. Партнёрская монетизация — только через `/go/{partner}`.

> **Статус: bootstrap.** Это форк рабочего движка **Georgia Guidebook** (`georgia-guidebook`):
> система сохранена целиком, грузинский контент снят, коллекции пустые (сайт собирается
> пустым по дизайну). **Контент и перемоделирование под Хорватию** (enums регионов/городов,
> SPEC, тексты, i18n) — **следующий шаг**, в этом коммите не делается. Грузинские планы,
> брифы и память оставлены как шаблоны в `docs/_georgia-reference/`.

> **Домен** `croatiaguidebook.com` — **плейсхолдер** (ещё не куплен); реальный задаётся на деплое.

## Стек

- **Astro 6** (полная статика SSG, без React) + **Content Collections**
- **Tailwind CSS 4**; дизайн-токены — только из `src/styles/tokens.css`
- Хостинг **Cloudflare Workers (Static Assets)** + Worker-роут `/go/` (`worker/index.ts`)
- Изображения webp/avif ≤ 200 КБ через предгенерацию вариантов (`npm run build:covers`)
- PWA (установка + оффлайн); одна аналитика

## Команды

| Команда | Что делает |
|---|---|
| `npm run dev` | dev-сервер |
| `npm run build` | прод-сборка (обязана быть без ошибок/предупреждений) |
| `npm run check` | `astro check` — типы и схемы контента |
| `npm run lint` / `npm run format` | ESLint / Prettier |
| `npm test` | контент-гейты: enums · parity · photos · interlinks |
| `npm run new -- <тип> <slug> --title "…"` | скаффолд материала + DoD-чеклист |
| `npm run qa` | финальный GO/NO-GO: чистая сборка + гейты + аудит `dist/` |

## Документация

1. **`SPEC.md`** — продукт, ниша, шаблоны, контент-модель (источник правды по продукту; перемоделируется под Хорватию на следующем шаге).
2. **`CLAUDE.md`** — железные правила работы в репозитории.
3. **`CONTENT_GUIDE.md`** — гайд по созданию контента.
4. **`SETUP-GUIDE.md`** — подключение внешних агентов/плагинов (my_agents, ui-ux-pro-max) как плагинов Claude Code.
5. **`docs/memory/`** — проектная память Хорватии. **`docs/_georgia-reference/`** — архив движка-донора (шаблоны).

> Форкнут из движка `georgia-guidebook`. Внешний тулинг подключается как плагины Claude Code — см. **`SETUP-GUIDE.md`** (в репозиторий не клонируется).
