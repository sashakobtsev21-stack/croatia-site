## 1. Почему Хорватия — и главный риск

[РЕШЕНО] croatia-site — **форк проверенного движка** (georgia-guidebook, через паттерн albania-site/montenegro-site) на новую страну. Движок страна-агностичен (контент-модель, кластеры, `/go/`-монетизация, QA, трёхъязычность) — переиспользуется целиком; перемоделируются только страна-специфичные слои.

**Тезис ниши:**
- **Член ЕС + ЕВРОЗОНА + ШЕНГЕН (всё с 01.01.2023)** — [ДАННЫЕ из анализа §2] уникальный козырь среди четырёх движков: ВНЖ/номад-виза Хорватии даёт доступ к ЕС/Шенгену, чего нет у Грузии/Албании/Черногории.
- **Сильный адриатический туризм** — [ДАННЫЕ §4] 21,3 млн прибытий / 108,7 млн ночёвок (2024), самая турозависимая экономика ЕС; платёжеспособная западная аудитория (плюс для EN/Tier-1/дисплея).
- **Цифровая номад-виза** — [ДАННЫЕ §2] до 18 мес + 0% хорв. НДФЛ на иностранный активный доход + Шенген + евро = сильнейший продающий нарратив для Tier-1 remote-workers.

> ### ⚠ ГЛАВНЫЙ РИСК (зафиксировать явно)
> [ДАННЫЕ §5 + верификация] В отличие от **тонкой Албании**, по Хорватии **EN-travel-контент НАСЫЩЕН и силён** — это зрелая «мекка»: крупные travel-медиа и матёрые блоги (nomadicmatt, lonelyplanet, ricksteves, frankaboutcroatia, chasingthedonkey, earthtrekkers) держат не только head-запросы, но и нишевые форматы «честный личный взгляд» и «off the beaten path». **Чистый туризм по Хорватии в лоб = отрицательный ROI.**
>
> **Следствия (приняты как рамка стратегии):**
> 1. **Моат «личного опыта» ОБЯЗАТЕЛЕН, а не опционален** (см. §5).
> 2. Ставка смещается на **релокейшн/номад-угол** (менее задавлен) и на **нишевые/глубокие темы**, где медиа поверхностны (паромы, конкретные острова, налоги/компания для иностранцев с личными цифрами).
> 3. **«Дешевизна» моатом быть НЕ может:** [ДАННЫЕ §3] Хорватия дороже Черногории (+13,8% CoL) и Албании (+16%), дешевле только Иберии. Value-угол слабый.

## 2. Ведущий язык: EN-first

[РЕШЕНО] **EN-first** (как Албания/Черногория). Архитектура движка без изменений: **en на корне `/`, ru → `/ru/`, uk → `/uk/`**, взаимные hreflang, x-default → en.

- EN — **приоритетный** язык: ключи, темы и приоритет берутся от **EN-интента** (в первую очередь — релокейшн/MONEY-кластер, §4).
- ru/uk — **вторичный** слой (дёшев в производстве через перевод движка), НЕ ведущий и НЕ источник приоритета. UK(укр) — структурно самый пустой язык в выдаче (дёшев как второй, но малый трафик); RU — брать информационно-опытным углом (там сплошной иммиграционно-риелторский лидген).
- `ka`/прочие языки не делаем.

## 3. Целевое гео: Tier-1 / Запад первично

[РЕШЕНО] **Первично — Tier-1 / Запад** (US / CA / UK / Зап. Европа / AU) через **EN**: дорогой RPM, западные туристы и удалёнщики-номады. **Вторично — СНГ + диаспора** через RU/UK.

**Принцип:** это сайт **ПРО Хорватию для аудитории НЕ в Хорватии**. Гео-таргет = **где читатели**, а не страна темы.

> **Подтверждение из анализа (§4):** растущие US и UK рынки (прямой US-доступ — United EWR–Дубровник, +EWR–Сплит лето-2026; плотный дешёвый UK-коридор), тогда как RU после 2022 практически обнулён. Это прямо поддерживает EN/Tier-1. Russian-аудитории для въезда нужна шенгенская виза (§2) → для RU угол смещается с «как въехать» на «релокация для уже легальных в ЕС/диаспоры».

## 4. Приоритет кластеров

[РЕШЕНО] **Релокация/номады подняты в приоритет** (менее сезонно, дороже RPM, Tier-1, рекуррентные партнёрки, И менее конкурентно, чем туризм по Хорватии) — противовес острейшей сезонности туризма (анализ §4).

**Порядок (EN-first):**

1. **Планирование** (верх воронки, ведёт трафик вниз; ETIAS/EES — горячая плохо покрытая тема)
2. **MONEY: аренда авто / страховка / eSIM** (транзакционные, AIO-стойкие)
3. **Релокация / Номады** (EN-first: cost of living, регистрация d.o.o., налоги/ВНЖ ЕС, **Croatia digital nomad visa**) — приоритетный денежный кластер
4. **Достопримечательности / Города / Маршруты** (TRAFFIC) — **только глубина/личный опыт/ниша**, НЕ конкурировать с медиа в лоб по общим запросам («things to do in Dubrovnik» забит)
5. **Еда**
6. **Развлечения / события** (фестивали как SEO-магнит; тикетинг-партнёрок нет — §6 анализа)

Детальный кластерный план с целевыми EN-запросами и денежными связками — **анализ §5/§6**.

> **Рекомендация по охвату (из верификации §5).** Не делать широкий страновой travel-сайт. Реалистичный продукт Хорватии — **узкий пакет ~15–30 страниц строго в MONEY/Релокация**: DNV-гайд (числа с MUP), `d.o.o.`/налоги для иностранца (личные цифры), living-in-Split/Zagreb (реальный бюджет), EU/Schengen-угол; туризм — длинный хвост-довесок (паромы, острова, логистика), не ядро.

> **YMYL-дисциплина (релокация/номады).** Визы/налоги/ВНЖ — чувствительные темы. Только офиц. источники (mup.gov.hr / porezna-uprava.gov.hr / PwC / EC) + видимая дата + дисклеймер. Номад-визу подавать как **18 мес** (с MUP), а не «3 года» из вторичных обзоров; ETIAS — **€20, старт Q4 2026**; EES — с 10.04.2026; для RU — «нужна шенгенская виза». Расхождения источников — анализ §2 и Приложение B.

## 5. Моат [ОТКРЫТО — главная развилка]

[РЕШЕНО 2026-06-24] **Владелец в Хорватии не живёт, поэтому широкий travel без моата не масштабируем.** Конкуренция **высокая** (выше тонкой Албании и средней Черногории) — рынок-«мекка»:
- EN-travel — стена брендов + зрелых блогов с личным опытом, заняты даже нишевые форматы;
- EN-релокация — first-hand-лидеров не один (expatincroatia.com), а рой опытных экспатов-консультантов (chasingthedonkey, sandinmycurls, dispatcheseurope, breezaabroad, relocationcroatia) — окно реально, но **уже заселяется**;
- «дешевизна» моатом быть не может (Хорватия дороже балканских сиблингов, §3).

[ОЦЕНКА] → **чистой свежести и курации НЕ хватит** (в отличие от Албании, где это работало). Для широкого travel нужен **реальный заменитель личного опыта**:
- сезонная база на побережье (владелец/доверенное лицо живёт летом), **или**
- местный стрингер/контрибьютор (Сплит / Загреб / Истрия).

**Решение владельца 2026-06-24:** до появления живого моата (поездка/стрингер/локальный контрибьютор) не наращивать broad-travel как ядро. Ближайший контент-курс — **relocation/money first**: digital nomad visa, cost of living, taxes, d.o.o., residence permits, living in Split/Zagreb, banking/insurance/eSIM. Broad-travel остаётся как уже опубликованный фундамент, перелинковка и long-tail, но не главный производственный приоритет.

Без решения по моату узкий релокейшн-клин ещё исполним (на козыре EU/Schengen + свежих числах с MUP против неактуализированных медиа), но широкий контент рискует упереться в «ещё один свежий компилятор» против осевших экспатов.

## 6. Монетизация

[РЕШЕНО] Модель **как серия (Грузия/Албания/Черногория)**: диверсифицированная **партнёрка через `/go/`** (доминирующий канал) + прямые размещения локалов + (для EN/Tier-1 Хорватии — опционально) дисплей. `partners.json` **переиспользуется** (Trip.com, SafetyWing, Airalo, EKTA, Localrent, Aviasales — все работают в Хорватии) **с 3 правками**: починить `trip-tours` URL (зашит `kwd=Georgia`), восстановить трекинг EKTA (сейчас прямой URL без атрибуции), сверить фактические ставки.

**Что добавить под Tier-1/EN** (подтверждено покрытие, анализ §6):
- **DiscoverCars** — приоритет авто-канала (70% прибыли проката, cookie 365 дн; покрытие ZAG/SPU/DBV);
- **GetYourGuide / Viator** — туры Дубровник/Сплит/Плитвице/Хвар (~8%);
- **Чартер яхт** — крупная HR-ниша с высоким чеком (Click&Boat / Sailo / Boataround / SamBoat — сверить программы);
- **Booking** — отели побережья (опц., отдельно от trip-hotels);
- опц. **Wise** (релоканты, CPA), **Saily** (eSIM, дешевле для EN).
- Сегментация `/go/` по языку: EN → DiscoverCars/SafetyWing/GetYourGuide/Booking/Wise; RU/UK → Localrent/EKTA/Aviasales/Airalo (**SafetyWing не для поездок в РФ/BY → СНГ через EKTA**).

**События/тикетинг:** [ДАННЫЕ §6] у локальных платформ (ulaznice.hr, Entrio.hr, Ultra Europe, INmusic) **публичных affiliate/API нет** — афиша фестивалей (Ultra Europe Split, Dubrovnik Summer Festival, INmusic Zagreb) как **SEO-магнит**, монетизация сопутствующим (жильё/авто/туры).

**Дисплей [ОТКРЫТО]:** для **EN-first Хорватии** реалистичен как 2-я нога (Tier-1 RPM, путь AdSense → Ezoic → Mediavine → Raptive; Raptive — 25k показов + большинство трафика US/CA/UK/AU/NZ). Цена — расширение строгого CSP движка, тяжёлый JS, риск CLS из 0. [ОЦЕНКА] **Рекомендация: на старте без дисплея, пересмотреть при достаточной доле Tier-1.**

**Ориентир дохода:** [ОЦЕНКА] зеркалит модель серии (партнёрка через `/go/` доминирует + рекуррент релокации/SafetyWing/Wise + потенциал Tier-1 дисплея). Плюс Хорватии — высокий Tier-1 RPM и платёжеспособная аудитория ЕС. **Абсолютные числа не фиксируем** — нет домена/GSC/трафика; вдобавок **пробиться в трафик ДОРОЖЕ из-за конкуренции** (войти стоит дороже, чем в Албании/Черногории). **Уточняется после запуска** (GSC + кабинеты партнёрок по SubID).
