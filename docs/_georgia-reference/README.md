# `_georgia-reference` — архив проекта Georgia Guidebook (ШАБЛОНЫ)

> ⚠️ Это **справочный архив** материалов рабочего движка Georgia Guidebook, на котором
> форкнут croatia-site. Содержимое относится к **Грузии** и оставлено здесь как
> **шаблоны/образцы** для следующего шага — перемоделирования под Хорватию.
> Это **НЕ актуальная документация** croatia-site и в продакшен/сборку не попадает.

## Что здесь лежит и как использовать

| Файл / папка | Что это (по Грузии) | Как переиспользовать под Хорватию |
|---|---|---|
| `CONTENT_PLAN.md` | План тем/разделов контента | Образец структуры плана — переписать под гео и спрос Хорватии |
| `GROWTH-MONETIZATION-ANALYSIS-*.md` | Анализ роста и монетизации | Методология та же — пересчитать рынок/партнёрки под Хорватию |
| `ROADMAP.md`, `ROADMAP-GROWTH-2026.md` | Роадмапы доработок | Шаблон роадмапа фаз |
| `PROGRESS.md`, `HANDOFF.md` | Журнал статуса и снимок «где остановились» | Формат ведения прогресса — начать с чистого листа |
| `AUDIT.md` | Сводный аудит сайта | Чек-лист аудита как образец |
| `TODO-affiliates.md` | Бэклог по партнёркам | Список партнёрок — пересобрать под доступные в Хорватии |
| `INSTRUKCIYA-KONTENT.md`, `KALENDAR.md` | Инструкция/календарь для владельца | Формат рабочего процесса |
| `briefs/` | Брифы статей (вход конвейера) | Образцы брифов — писать новые под Хорватию |
| `memory/` | Проектная память Georgia Guidebook | История решений движка; новая память — в `docs/memory/` |

## Системные доки (НЕ здесь — в корне репозитория, актуальны)

`SPEC.md`, `CLAUDE.md`, `CONTENT_GUIDE.md`, `SETUP-GUIDE.md` остались в корне как
**рабочие** документы движка. Грузинская специфика внутри них (enums регионов/городов,
тексты, домен) перемоделируется под Хорватию на следующем шаге, а не в этом архиве.
